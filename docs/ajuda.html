Ajuda Rapida

Evo XTTS V2

Guia simples para instalar, abrir e usar a interface local no Windows.

Abrir Interface Abrir API Docs

Fluxo simples

1

Instalar

Clique em Instalar XTTS.bat.

2

Adicionar voz

Coloque um arquivo .wav dentro da pasta voices.

3

Abrir

Clique em Abrir XTTS.bat.

4

Gerar audio

Digite o texto, escolha a voz e gere o audio no navegador.

Recomendacoes

  • Use arquivo .wav
  • Prefira 6 a 30 segundos
  • Uma pessoa so falando
  • Sem musica ao fundo
  • WAV e o formato mais recomendado
MP3 e opcional

Se quiser exportar MP3, instale ffmpeg.

Arquivos importantes

  • Instalar XTTS.bat: prepara o ambiente
  • Abrir XTTS.bat: inicia a aplicacao
  • ui/index.html: interface web
  • voices/: pasta das vozes

Se algo der errado

  • Confirme que existe um .wav dentro de voices
  • Execute Instalar XTTS.bat novamente
  • Se a tela travar, use Ctrl+F5
  • Se o navegador nao abrir, acesse http://localhost:8881/
