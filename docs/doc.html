Quick Guide

Evo XTTS V2

Simple instructions to install, launch, and use the local Windows interface and API.

Open Interface Open API Docs

Quick Start

1

Install

Run install.bat to prepare the environment and dependencies.

2

Add A Voice

Place at least one .wav file inside the voices folder.

3

Launch

Run start.bat and wait for the browser to open.

4

Generate Audio

Enter text, choose language, voice, emotion, and format, then generate full audio or streaming audio.

Recommended Voice Samples

  • Use .wav files
  • Prefer 6 to 30 seconds
  • One speaker only
  • No background music
  • Clean and consistent recording volume
  • WAV is the recommended output format
MP3 is optional

If you want MP3 output, install ffmpeg.

Important Files

  • install.bat: source install entry point
  • start.bat: source launch entry point
  • ui/index.html: web interface
  • voices/: cloned voice samples
  • README.md: full project documentation

If Something Goes Wrong

  • Confirm there is at least one .wav file inside voices
  • Run install.bat again if dependencies are missing
  • Use Ctrl+F5 if the interface looks stale
  • If the browser does not open, visit http://localhost:8881/ manually
  • Check http://localhost:8881/health to confirm the API is ready
